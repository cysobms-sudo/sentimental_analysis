 📊 تحلیلگر احساسات بازار مالی

اخبار، گزارش‌ها و تحلیل‌های مالی را وارد کنید تا احساسات بازار را بسنجید

 💡 کاربرد
تحلیل اخبار بورس
 بررسی گزارش‌های سودآوری
 تحلیل‌های اقتصادی
 توصیه‌های سرمایه‌گذاری

 🚀 نحوه استفاده
1. متن خبر یا تحلیل مالی را در کادر وارد کنید
2. روی دکمه تحلیل کلیک کنید
3. نتیجه تحلیل را با فونت درشت مشاهده کنید

📂 آپلود فایل
می‌توانید فایل متنی (TXT) خود را آپلود کنید تا برنامه خط‌به‌خط آن را تحلیل کند.
